## **7. Repair Rules**
Repair cost is 10% of base item cost x 0.25 x Quality Multiplier

| Damage Level               | Time Multiplier |
| -------------------------- | --------------- |
| Minor (dull, bent)         | x0.25           |
| Moderate (cracked, warped) | x0.5            |
| Severe (broken, magical)   | x0.75           |

***

## **Example: Creating a Longsword**, 150 gp, 43.2 days

*   Forge: **Excellent**  (+4, x0.8)
*   Smith: **Weaponsmith**  (+2, x0.9)
*   Material: **High-quality Steel**  (+2)
*   Base weapon: **Longsword** (15 gp, 5 days)

    Quality: 14 on d20, +4(Forge) +2(Smith) +2(Material) = 22 for a Masterwork (x10 multiplier)  
    Cost: 15gp x 10 = 150 gp  
    5 days x 0.8(Forge) x 0.9(Smith) x 1.2(Material) x 10(Masterwork) =  43.2 days
