## **7. Repair Rules**
Repair cost is 10% of base item cost x 0.25 x Quality Multiplier

| Damage Level               | Time Multiplier |
| -------------------------- |:---------------:|
| Minor (dull, bent)         | x0.25           |
| Moderate (cracked, warped) | x0.5            |
| Severe (broken, magical)   | x0.75           |

## **Example: Excellent Longsword**  

*   Base cost: Longsword is 15gp
*   Forge: **Excellent** (+2, x0.8)
*   Smith: **Weaponsmith** (+1, x0.95)
*   Material: **High-quality Steel** (+1, x1.1)
*   Base Time: **Full Weapon** 3 on d6 + 2 days = 5 days

Quality: 14 on d20, +2(Forge) +1(Smith) +1(Material) = 18 {**Excellent**}  
Cost: 15gp x 2(Excellent) = 30gp  
Time: 5 days x0.8(Forge) x0.95(Smith) x1.1(Material) x2(Quality) =  8.36 days  

## **Example: Fine Dagger**  

*   Base cost: Dagger is 2gp
*   Forge: **Superior** (+1, x0.9)
*   Smith: **Blacksmith** (0, x1)
*   Material: **Standard Steel** (0, x1)
*   Base Time: **Small Tool** 1 day

Quality: 13 on d20, +1(Forge) = 14 {**Fine**}  
Cost: 2gp x 1.5(Fine) = 3gp  
Time: 1 day x0.9(Forge) x1(Smith) x1(Material) x1.5(Quality) = 1.35 days  

## **Example: Legendary Adamantium Battleaxe**  

*   Base cost: Battleaxe is 6gp
*   Forge: **Legendary** (+3, x0.7)
*   Smith: **Master Smith** (+2, x0.85)
*   Material: **Adamantium** (+3, x1.5)
*   Base Time: **Full Weapon** 5 on d6 + 2 days = 7 days

Quality: 19 on d20, +3(Forge) +2(Smith) +3(Material) = 27 {**Legendary**}  
Cost: 6gp x 10(Legendary) = 60gp + material  
Time: 7 days x0.7(Forge) x0.85(Smith) x1.5(Material) x10(Quality) =  62.5 days
